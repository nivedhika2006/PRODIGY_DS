# PRODIGY_DS 🎓

## Prodigy InfoTech - Data Science Internship
This repository contains all 5 tasks completed during the 
Prodigy InfoTech Data Science Internship.

## Task 01 - Population Distribution Visualization 📊
- Created bar charts and histograms to visualize population distribution
- Dataset: World Bank Population Data
- Tools: Python, Pandas, Matplotlib, Seaborn

## Task 02 - Titanic EDA & Data Cleaning 🚢
- Performed data cleaning and exploratory data analysis
- Explored relationships between variables and identified patterns
- Dataset: Titanic Dataset (Kaggle)
- Tools: Python, Pandas, Matplotlib, Seaborn

## Task 03 - Decision Tree Classifier 🌳
- Built a decision tree classifier to predict customer purchases
- Dataset: Bank Marketing Dataset (UCI)
- Tools: Python, Pandas, Scikit-learn, Matplotlib

## Task 04 - Sentiment Analysis 💬
- Analyzed sentiment patterns in social media data
- Visualized public opinion using WordClouds and charts
- Dataset: Twitter Entity Sentiment Dataset (Kaggle)
- Tools: Python, Pandas, Matplotlib, WordCloud

## Task 05 - Traffic Accident Analysis 🚗
- Analyzed traffic accident data to identify patterns
- Visualized accident hotspots and contributing factors
- Dataset: US Accidents Dataset (Kaggle)
- Tools: Python, Pandas, Matplotlib, Seaborn
